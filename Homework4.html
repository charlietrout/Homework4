The getRMSE() function computes the Root Mean Square Error (RMSE) between actual and predicted values. It handles missing values (NA) using optional arguments (…) to ensure flexibility in data processing. 1.

getRMSE <- function(resp, pred, ...) {
  # Calculating RMSE
  rmse <- sqrt(mean((resp - pred)^2, ...))
  return(rmse)
}

The getMAE() function computes the Mean Absolute Error (MAE) between actual and predicted values. Like getRMSE() above, it accommodates missing values with (…) if need be and provides a robust metric for evaluating prediction accuracy. 3.

getMAE <- function(resp, pred, ...) {
  # Calculating MAE
  rmse <- mean(abs(resp - pred), ...)

The getMetrics() function acts as a wrapper to compute multiple evaluation metrics (such as RMSE and MAE) for comparing actual and predicted values. It checks inputs for validity and uses optional arguments (…) to handle missing values appropriately if need be. 5.

getMetrics <- function(actual, predicted, metrics = c("RMSE", "MAE"), ...) {
  # Check if actual and predicted are numeric vectors
  if (!is.atomic(actual) || !is.numeric(actual) || !is.atomic(predicted) || !is.numeric(predicted)) {
    message("Error: 'actual' and 'predicted' must be numeric vectors.")
    return()
  }
  results <- list()
  if ("RMSE" %in% metrics) {
    results$RMSE <- getRMSE(actual, predicted, ...)
  }
  if ("MAE" %in% metrics) {
    results$MAE <- getMAE(actual, predicted, ...)
  }
  return(results)
}

The queryNewsAPI() function queries the NewsAPI using parameters like search query (q), start date (from), and API key (apiKey). It retrieves news articles matching the specified criteria and parses the JSON response for further analysis. 3.

# Function to query News API
queryNewsAPI <- function(q, from, apiKey) {
  # Base URL for the News API
  base_url <- "https://newsapi.org/v2/everything"
  
  # Construct query parameters
  query_params <- list(
    q = q,
    from = from,
    apiKey = apiKey
  )
  
  # Make GET request
  response <- httr::GET(base_url, query = query_params)
  
  # Parse JSON content
  parsed_response <- fromJSON(rawToChar(response$content))
  
  return(parsed_response)
}
